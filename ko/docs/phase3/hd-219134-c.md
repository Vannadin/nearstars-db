# HD 219134 c — Phase 3 Synthesis

HD 219134 c 는 시스템의 두 번째 transit 행성으로, 밝고 조용한 K3 V 왜성
HD 219134 주위에서 b 바로 바깥에 있습니다. b 처럼 측정된 진질량과
측정된 반지름을 모두 가지고 있고 (Gillon et al. 2017, Spitzer 측광),
M = 4.36 ± 0.22 M⊕, R = 1.511 ± 0.047 R⊕, 작은 이심률 e = 0.062 의
0.064 AU 6.765 일 궤도입니다. bulk 밀도는 시스템에서 가장 높아
ρ ≈ 7.0 g/cc — b 보다도 밀도가 높고, volatile envelope 없이 확고히
암석/철 풍부합니다. b 보다 차갑습니다 — TEPCat 은 T_eq ≈ 782 K 를
제시하고, 프로젝트 공식 T_eq = 278.3 · L^0.25 / √a (L = 0.264) 는 zero
albedo 에서 789 K 를 줍니다.

**NearStars 시나리오 선택은 어두운 basalt 표면과 희미한 산화 / sub-melt
accent 를 가진, 뜨겁고 매우 밀도 높은 무대기 암석 super-Earth 입니다 —
b 보다 차갑고 basalt 용융점 아래라 substellar 마그마 cap 은 없지만,
여전히 구워지고 대기가 빈약한 암석입니다.** 789 K 와 지구의 ~64 배
일사에서 c 는 부분용융 임계 아래지만 물을 유지하기엔 너무 뜨겁습니다.
극단적인 측정 밀도 (7.0 g/cc) 가 다시 두꺼운 volatile envelope 을
배제합니다. 대안 — 유지된 얇은 CO₂ 대기 — 은 후속 항목에 cfg 변형으로
보존합니다.

## 표면 합성

HD 219134 c 는 시스템에서 가장 밀도 높은 행성이며 transit 하는 두 암석
super-Earth 중 둘째입니다. 세 가지 앵커가 합성을 고정합니다.

1. **측정된 질량과 반지름.** Gillon 2017 의 Spitzer transit 이 R =
   1.511 ± 0.047 R⊕ 를 주고, transit + RV 결합 해가 M = 4.36 ± 0.22 M⊕
   를 줍니다. bulk 밀도 ρ ≈ 7.0 g/cc (유도) 는 시스템 어떤 행성보다도
   높습니다 — b 보다도 밀도가 높고, volatile envelope 이 들어갈 자리
   없이 확고히 암석/철 풍부합니다. Gillon 2017 은 b/c 쌍을 알려진
   transit super-Earth 중 가장 밀도 높은 축으로 강조합니다.

2. **일사 64.5 S⊕, T_eq ≈ 789 K.** b 보다 차갑지만 여전히 강렬하게
   뜨겁습니다. substellar 무대기 정점은 ~970 K (유도) 에 도달하는데,
   이는 basalt solidus (~1300 K) 아래입니다 — 그래서 b 와 달리 c 는
   substellar 마그마 cap 이 없습니다. lava world 가 아니라 구워지고
   메마른 암석입니다.

3. **대기 손실.** b 와 마찬가지로 어떤 원시 H/He envelope 도 오래전에
   광증발했고 (Owen & Wu 2017), 극단적 밀도가 두꺼운 envelope 이 없음을
   확인합니다. 2차 대기는 가까운 XUV flux 아래 789 K 에서 미미합니다.

**색 선택.** 4858 K K3 V 오렌지 조명 아래에서 c 의 표면은 어두운 구워진
basalt 지각 `#3a3026` 으로 렌더링됩니다 — b 의 용융 임박 `#2a2422`
보다 약간 갈색이고 밝아, 더 차가운 미용융 상태를 반영합니다. accent
`#8a3a1a` 은 용암 glow 가 아니라 구워진 산화철 / regolith 적갈색입니다
(tie-break, 흥미로움 우선). c 는 용융점 아래라 독특한 feature 가 빛나는
substellar cap 이 아니라 산화 얼룩입니다.

**조석 고정 하의 지형.** 표면 용융이 없어 c 의 외관은 강하게 일사를 받는
수성에 더 가깝습니다 — 시스템 나이에 걸쳐 누적된 충돌 cratered basalt
평원이 구워지고 volatile 이 제거되며, 크레이터를 지울 마그마 재포장이
없습니다. substellar 반구는 가장 뜨겁지만 여전히 고체이고, antistellar
반구는 차갑고 어두운 regolith 로 cold-trap 이 없습니다.

## 대기 합성

c 의 대기 추론은 이론적이지만, b 처럼 측정된 밀도로 강하게 제약됩니다.
cfg 는 **무대기** 해석을 채택합니다.

**압력 선택.** 표면 압력 0 Pa (무대기). 7.0 g/cc 에서 c 는 시스템에서
가장 밀도 높은 행성입니다 — volatile envelope 없는 순수 암석과 철이고,
Owen & Wu 2017 의 광증발 physics 가 0.064 AU 의 4.4 M⊕ core 를 core-only
영역에 둡니다. 789 K 에서 2차 대기는 미미해 cfg 는 무대기를 선택합니다.
c 는 b 보다 차가워 exosphere 로의 표면 outgassing 마저 약하므로, 사실상
맨 암석입니다.

**조성.** 정수압 대기 없음. 어떤 표면 기원 exosphere (Schaefer & Fegley
2009 스케일링) 도 b 의 것보다 희미한데, 더 낮은 substellar 온도가
silicate 휘발을 억누르기 때문입니다. column 밀도는 Rayleigh 임계 한참
아래라 보이는 limb haze 가 없습니다.

**하늘 외관.** 대기가 없으니 하늘은 균일하게 검습니다. HD 219134 가 c 의
dayside 하늘을 각지름 ~6.0° 로 지배하고 (지구에서 본 태양 각 크기의 약
12 배), 따뜻한 오렌지 disk 가 표면을 ~64× 지구 정오 밝기로 적십니다.
안쪽 행성 b 가 합 시 밝은 점으로 보이고, 바깥 행성들은 더 희미합니다.
limb 는 disk 에서 우주로의 깔끔한 가장자리입니다.

## 자전 & spin 합성

0.696 M☉ 호스트 주위 0.064 AU 의 6.765 일 궤도 주기는 시스템 나이보다
훨씬 짧은 조석 고정 timescale (Hut 1981) 을 주므로, 1:1 회전-궤도 공명이
완전히 확립되었고 황도경사각도 0 까지 damping 되었습니다.

**회전-궤도 상태.** curated 이심률은 0.062 — 작지만 0 이 아닙니다. 이는
1:1 이 유일한 안정 상태가 되는 e ≈ 0.01 경계 바로 위라, 수성 (e = 0.205)
유사로 3:2 공명 trap 이 원칙적으로 가능합니다. 다만 조밀한 다행성 사슬에서
느린 이심률 damping 은 1:1 을 유력한 종착점으로 선호합니다. cfg 는 1:1
(substellar 점 고정) 을 채택하고 3:2 대안은 후속 항목에 cfg 변형으로
기록합니다.

**KSP 구현 노트.** 자전 주기 = 궤도 주기 = 6.765 일 (584 496 s)
입니다. Kopernicus 에서 body 의 `rotationPeriod` 는 궤도 `period` 와 초
단위로 동일하게 설정합니다.

**계절 없음.** obliquity 가 0 까지 damping 됨. 작은 이심률 (e = 0.062)
은 궤도에 걸쳐 peri-에서-apo 일사 범위 S = (1 ± e)⁻² × 64.5 ≈ 57 → 73 S⊕
를 주지만, substellar 점이 고정되어 이는 계절 날씨가 아니라 dayside
온도의 약간의 변조로 나타납니다. e = 0.062 에서 조석 가열은 적당하고
(Bolmont 2020 스케일링), 표면 에너지 예산은 일사 지배입니다.

## 비주얼 스타일

표면과 대기 결정을 종합하면 다음과 같습니다.

- **전역 외관 (궤도 뷰).** 구워졌지만 녹지 않은 어두운 암석 super-Earth
  로, 어두운 basalt 평원과 희미한 산화 적갈색 얼룩을 가지며 nightside
  로 갈수록 검어집니다. (b 와 달리) 빛나는 substellar cap 은 없습니다 —
  c 는 용융점 아래입니다.
- **Dayside.** 뜨거운 (~970 K substellar) 구워진 basalt, tint `#3a3026`,
  `#8a3a1a` 산화철 accent 얼룩. 매끄러운 용암 평원이 아니라 수성 유사
  cratered 지형.
- **Nightside.** 차갑고 (~130 K) 어두움. dayside 와 같은 tint 이지만
  조명 없음. 자매 행성 반사광으로만 보임.
- **Terminator band.** 대기 scattering 이 없어 또렷함. 크레이터 림과
  능선이 스치는 오렌지 빛에 긴 그림자를 드리움.
- **대기 haze.** 보이지 않음. 무대기. limb 는 깔끔한 가장자리.
- **하늘의 별.** HD 219134 가 c 에서 ~6.0° 를 차지 — 지구에서 본 태양 각
  크기의 약 12 배. 따뜻한 오렌지 (4858 K), 표면을 ~64× 지구 정오 밝기로
  조명.
- **하늘의 자매 행성.** b (안쪽, 0.0382 AU) 가 내합 시 가장 두드러진
  이웃으로 거의 동일 평면. 따뜻한 RV-only 행성들과 먼 거인 h 는 밝은
  점으로 보임.

  타깃입니다. MIRI eclipse 가 dayside brightness temperature 와 Bond
  albedo 를 측정해 표면 tint 를 정밀화합니다.
- **cfg 변형: 유지된 얇은 CO₂ 대기.** 얇은 고분자량 대기 (≲ 0.1–1 bar
  CO₂) 가 형식적으로 여전히 가능합니다. 얇은 hazy 대기를 가진 cfg
  변형을 "유지된 대기" 해석을 위해 보존합니다.
- **cfg 변형: 3:2 회전-궤도 공명.** e = 0.062 에서 c 는 1:1-전용 경계
  바로 위입니다. 수성 유사 3:2 변형은 천천히 표류하는 최고온 경도로
  렌더링됩니다. Phase 3.5 대안으로 기록.
- **표면 광물학.** hex 코드는 tie-break 선택입니다. 향후 열복사 분광이
  이들을 정밀화합니다.
