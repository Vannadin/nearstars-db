# HD 69830 c — Phase 3 Synthesis

HD 69830 c 는 Lovis 외 2006 (HARPS, 4 년 캠페인) 이 발견한 세 개의
넵튠 질량 시선속도 행성 중 중간에 있습니다. P = 31.56 ± 0.04 일,
Msini = 11.8–12.1 M⊕ (Lovis 2006 / Tanner 2019 refit), a = 0.186 AU,
e = 0.13, ω = 221°. 모성이 L = 0.60 L☉ 이므로 c 의 일조량은 지구의
~17 배, A = 0.3 본드 알베도에서 평형 온도는 519 K — 시스템 5.7 ±
1 Gyr 나이 안에 일차적 Neptune 급 H/He 외피가 편안히 유지되는
온도입니다. 통과 검출은 없습니다. Bonfils 2007 과 후속 측광이
예측된 ephemeris 를 배제하기 때문에 관측으로 제약되는 것은 Msini 와
궤도 요소뿐입니다. 궤도 장반경 0.186 AU 는 c 를 0.7–0.8 AU 의
따뜻한 먼지 벨트 내경 (Beichman 2005) 보다 ~3.8 배 안쪽에 두며,
c 는 먼지 인구를 가로지르지 않고 그 안쪽에 단단히 자리합니다.

**NearStars 시나리오 선택. 따뜻한 외피-유지 Neptune analog 으로,
총 질량의 ~25% 에 해당하는 두꺼운 H₂/He 대기, 물 풍부 얼음 내부,
그리고 행성의 결정적 시각 시그니처를 만드는 chromophore 함유
광화학 haze 층을 가진 sub-Neptune 에서 mini-Neptune 급의 천체입
니다.** 28 cfg 픽 중 12 는 canonical-aligned (궤도 + 질량 파라미터,
유도된 일조량 / 평형 온도), 16 은 tie-break (외피-유지 mini-Neptune
윈도우의 반지름 / 밀도, 광화학 chromophore 색조, 적당한 조석-자전
결합 하의 구름 형태) 입니다. documented divergence 는 없습니다.
오래된 G8V 주위 17 S⊕ 의 12 M⊕ 행성에서 외피-유지 해석이
canonical 이며, 광증발 valley 바닥 위로 편안히 자리하고 Rogers
2021 의 외피-유지 위치에 들어 있습니다.

## Surface synthesis

HD 69830 c 는 지질학적 의미의 표면이 없습니다. 외피-유지 mini-
Neptune 시나리오는 c 를 광증발 valley 위로 단단히 놓습니다. 12.1 M⊕,
17.3 S⊕ 에서 c 는 Fulton 2017 과 Rogers 2021 의 외피-유지 위치
안에 편안히 자리합니다 — XUV 구동 질량 손실이 Gyr 시간 스케일에
걸쳐 substantial 한 외피를 박탈하는 ~5 M⊕ 임계값 위로 한참. 오래된
G 왜성 호스트라도 마찬가지입니다. cfg 는 canonical Neptune-analog
내부 구조를 채택합니다. 작은 (~3 M⊕) 암석/철 코어, 그를 둘러싼
깊은 (~6 M⊕) 초임계 H₂O / NH₃ / CH₄ 얼음 mantle, 그 위의 두꺼운
(~3 M⊕, 총 질량의 ~25%) H₂/He 외피. 렌더러가 보는 "표면" 은
상부 대류권 ~0.1 bar 의 구름 / haze 데크입니다.

외피-유지 해석은 질량-일조량 다이어그램의 이 지점에서 canonical
입니다 — b (10 M⊕, 97 S⊕, marginal 증발 위치) 에 중요했던 Lovis
2006 §5 의 bare-core 모호성은 c 에는 적용되지 않습니다. 오래된
G8V 주위 0.186 AU 에서의 광증발 시간 스케일은 b 의 0.078 AU 보다
수 Gyr 더 깁니다. 그리고 host XUV 광도 (log L_X ≈ 26.7, Merloni
2024) 가 시스템 나이에 걸쳐 12 M⊕ 외피로부터 의미 있는 질량 손실을
구동하기에 너무 약합니다. cfg 는 따라서 stripped-core 변형 없이
canonical 외피-유지 Neptune 을 선택합니다.

cfg-variant 컷어웨이 렌더러를 위한 개념적 내부 색은 Neptune-analog
깊은 mantle 을 인코딩합니다. 초임계 얼음 mantle 주체의 `#5a7060`
깊은 청회색, 그리고 암석 코어에서 오는 대류 plume 이 더 따뜻한
물질을 외피 바닥 쪽으로 가져오는 곳의 `#80a890` 더 따뜻한 녹청색
액센트. 이는 순수한 tie-break 으로, cfg 의 내부 컷어웨이 뷰용입니다.
관측이 내부 지오메트리를 제약하지 않습니다. 초임계-얼음 mantle 아래,
내부 암석/철 코어 조성은 모성의 금속도 스케일링 ([Fe/H] = −0.05) 을
따르며, Sol-analog 가정에 대해 약간 Fe-결핍이지만 인구-전형 범위
안입니다.

외피의 깊은 바닥은 얼음-mantle / 외피 경계 (압력 ~50 kbar, 가시
광자 탈출 깊이 훨씬 아래) 에서 ~3000 K 에 도달합니다. 렌더러의
내부 컷어웨이 뷰는 층 구조를 도식적으로 보여줍니다 — modest 한
조석 가열 (e = 0.13 으로 ~0.002 W/m²) 너머 내부 에너지 예산을
제약하는 관측 속성이 없습니다. modest 조석 가열은 Neptune 자체의
내적 열 플럭스 (~0.4 W/m²) 보다 한참 아래이므로 형성의 primordial
열과 방사성 붕괴에 비해 부차적입니다.

## Atmosphere synthesis

대기는 c 의 결정적 관측 가능 요소입니다. Moses 2013 의 따뜻한
Neptune 화학을 따르면, 520 K 평형 온도는 c 를 "따뜻한
Neptune" 영역에 놓습니다. CH₄ 가 지배적인 탄소 운반체 (더 뜨거운
T > 1000 K 의 CO 와 대조) 이고, NH₃ 가 하부 성층권에서 안정 (더
뜨거운 T 의 N₂ 와 대조) 하며, H₂O 응결이 외피 바닥의 초임계 혼합에
의해 억제됩니다. ~0.1 bar 의 가시 구름 데크는 NH₃ 얼음 / 하이드로
설파이드 (NH₄SH) 구름 입자와 상부 성층권의 광화학 황 함유 chromophore
haze 로 구성되며, Saturn 의 중위도 광화학 haze 의 밝은 올리브 /
노란 색조와 유사합니다.

cfg 는 약한 substellar / antistellar 비대칭과 함께 띠 zonal flow
형태 (`cloud_cover_fraction = 0.70`) 를 채택합니다. P_rot ≈ 0.84
P_orb (Hut 1981, Greenberg 2009 의 e-의존 균형) 의 pseudo-synchronous
회전 상태는 c 에게 ~26.5 일의 자전 주기를 줍니다 — Showman & Polvani
2011 zonal-jet 형태가 modest 한 띠 구조를 발달시키지만 강한
Saturn-Jupiter 대비는 아닐 정도로 느림. cfg 는 중위도를 가로지르는
~3 개의 교차하는 chromophore-rich (어두운 올리브-골드) 와
chromophore-poor (더 밝은 올리브-크림) 띠를 인코딩하며, 더 약한
열대 / 극지 대비를 둡니다.

대기 색조는 `#a8b885` 따뜻한 올리브-골드입니다 — Sudarsky 2003 의
Class III ("따뜻한 Neptune") 팔레트에 520 K 에서 Moses 2013 의 황
함유 탄화수소 haze 의 광화학 chromophore 기여를 더한 것. 모성이
태양보다 살짝 더 차갑기 (5385 K vs 5772 K) 때문에 인지된 색조가
더 따뜻한 / 더 붉은 쪽으로 이동하지만, 지배적인 신호는 별 조명보다
대기 chromophore 화학에서 옵니다. 구름 색조 `#c8d4a0` 은 띠 구조의
밝은 zone 으로, 궤도 스케일에서 교차하는 belt-zone 줄무늬로 가시
입니다.

**substellar 영역에서 본 하늘.** 구름 데크 위에서 별이 2.3° 디스크
로 하늘을 채우며 — 지구에서 본 태양 겉보기 지름의 약 4 배 — 태양
보다 살짝 더 붉은 크림-노랑 색조가 상부 대기를 변조합니다. 0.7–
1.5 AU 의 따뜻한 먼지 벨트는 c 의 외부로 4–8 배 떨어져 있으며,
c 의 시점에서 정렬 사건 동안 하늘을 가로지르는 희미한 호박색 띠로
투영됩니다 — b 에서보다 뚜렷이 더 밝지만 여전히 모성의 압도적
글로우 한참 아래. 0.63 AU 의 자매 행성 d 는 c 에서 opposition
(최근접 0.44 AU) 동안 0.03° 각지름으로 나타나, HD 69830 자체를
제외한 어떤 별보다도 밝습니다.

## Rotation & spin synthesis

5.7 ± 1 Gyr 에 걸친 c 의 31.56 일 궤도 주기 조석 damping 은 완전한
1:1 동기화에 도달하지 못합니다. 오래된 G 왜성 주위 0.186 AU 의
Neptune 급 행성에 대한 동기화 시간 스케일은 ≈ 10⁸ 년 (Heller 2011
스케일링) 으로 시스템 나이와 비슷하지만 의미 있게 짧지는 않습니다.
e = 0.13 에 대한 평형 spin 상태는 pseudo-synchronous (Hut 1981,
Greenberg 2009) 이며, 자전 주기 P_rot ≈ 0.84 × P_orb = 26.5 일로,
substellar 점이 관성 좌표계에서 ~5°/day 의 느린 드리프트를 갖습니다.
황도 경사도는 sync 행성의 완전한 0° 대신 ~5° 로 부분적으로 damping
되며, pseudo-sync 상태를 확립하는 동일한 e 구동 강제력에 의해
보존됩니다.

pseudo-sync 자전율은 가까운 조석 고정 hot Jupiter 의 강한 zonal-wind
영역 (Showman & Polvani 2011) 보다 훨씬 느리지만, 표면 좌표계 자전
율의 ~2–3 배만큼 적도 superrotation 을 구동할 정도로 빠릅니다. 결과
적인 띠 형태는 Saturn 의 강한 belt-zone 구조 (3 일 자전) 와 Neptune
의 더 약한 띠 (16 시간 자전이지만 차가움) 사이에 중간입니다 — c 는
가시 대기 동역학에서 Neptune-analog 보다 따뜻한 Saturn 에 더 가깝
습니다.

**KSP 구현 노트.** pseudo-sync cfg 에서 자전 주기 = 26.5 일
(2 289 600 s). canonical 대안 cfg 변형 (P_rot ≈ 16 h, Neptune-
analog 자유 회전자) 이 Open items 에 보존됩니다. 그 해석 하에서는
띠가 더 sharp 하고 substellar 드리프트가 더 빠르겠지만, pseudo-sync
균형 상태가 e = 0.13 의 오래된 행성에 대한 동역학적으로 예상되는
구성입니다.

**이심 궤도 일조량 사이클링.** e = 0.13 에서 별-행성 거리가 각
31.56 일 궤도 동안 0.162 AU (근일점) 와 0.210 AU (원일점) 사이를
변하며, 일조량이 22.9 S⊕ (근일점) 와 13.6 S⊕ (원일점) 사이를
사이클링합니다. ~50% 일조량 진폭이 상부 대기에 b 와 비교 가능한
thermal-tide 강제력을 구동하지만, 800 K 가 아닌 520 K 에서 haze
화학 반응이 덜 극적입니다 — cfg 는 궤도당 상부 구름 데크에서
peak-to-peak ~30 K 의 스윙을 예상합니다. cfg 는 애니메이션하지
않습니다. 더 깊은 내부 모델용 정보입니다.

## Visual styling

- **글로벌 외관.** 따뜻한 올리브-골드 디스크 (`#a8b885` 중심 색조,
  `#c8d4a0` 더 밝은 zonal 띠), HD 69830 b 시점에서 약 0.04° 겉보기
  지름 — c 에서 본 b 의 겉보기 크기와 비슷. c 외부 플레이어 시점
  에서는 궤도 뷰 스케일에서 분해 가능한 ~3 개의 chromophore-rich /
  chromophore-poor 줄무늬가 있는 띠 올리브-골드 구조가 보이며,
  b 의 균일한 haze 디스크보다 시각적으로 더 두드러집니다.
- **Dayside 디테일.** 가장 따뜻한 substellar 점에서의 chromophore
  화학에서 오는 modest substellar dome brightening (글로벌 색조
  대비 ~5%). dayside 는 substellar 점 근처의 `#c8d4a0` 에서
  terminator 의 `#a8b885` 까지 범위.
- **Terminator 밴드.** 부드러운 Neptune-analog 산란 프로필을 가진
  올리브에서 어두운 올리브-회색 (`#5a6850`) 으로의 gradient. sharp
  edge 없음.
- **Nightside.** 470 K 야측 구름 데크의 잔존 열 방출에서 오는
  어둑한 올리브-회색 글로우. 야측 IR 렌더링이 활성화된 cfg 변형
  에서만 보임.
- **대기 & 구름 구조.** 결정적 피처는 띠 zonal flow. 저-중위도의
  chromophore-rich 어두운 띠가 더 밝은 chromophore-poor zone 과
  교차하며, 궤도 스케일에서 가시 반구를 가로지르는 ~3 개의 띠.
  띠는 pseudo-sync 회전과 함께 천천히 드리프트. 자세히 보면 미묘한
  파동 패턴이 가시.
- **하늘의 별.** HD 69830 이 c 의 하늘에서 2.27° — 지구에서 본
  태양 겉보기 지름의 약 4 배, 수성 궤도 내부에서 본 태양과 비슷한
  큰 크림-노랑 디스크.
- **하늘의 자매 행성.** c 에서 b 는 합 (a/b 분리 0.11 AU, b 의
  haze 디스크 반지름 ~16 600 km) 동안 각지름 ~0.03°, d 는 d 의
  최근접 (a/d 분리 0.44 AU, d 의 반지름 ~28 400 km) 동안 ~0.04°.
  d 가 더 먼 거리에도 더 큰 반지름 덕분에 c 의 시점에서 둘 중
  더 밝습니다.
- **c 시점에서 본 따뜻한 먼지 고리.** 0.7–1.5 AU 의 먼지 벨트가
  c 의 0.186 AU 궤도 외부로 4–8 배 떨어져 있습니다. c 에서는
  궤도 평면을 가로지르는 호박색 띠로 보입니다 — 가상의 벨트 외부
  관찰자 시점보다는 좁고 어둑하지만, b (벨트 안쪽으로 13 배)
  에서보다 뚜렷이 더 밝습니다. 근일점 정렬 동안 밝기가 피크에
  도달하지만 모성 한참 아래 머무릅니다.

## Open items for follow-up

- **통과 탐색 갱신.** 2026 년 현재 발표된 통과 검출 없음. CHEOPS /
  TESS short-cadence 모니터링이 grazing 지오메트리를 ruling in/out
  하고 진짜 반지름을 핀할 수 있습니다. c 에 대한 통과 검출은 질량-
  반지름 tie-break 을 해소하고 `radius_rearth` 를 3.46 R⊕ 에서
  측정값으로 업데이트할 것.
- **Pseudo-sync vs 자유 회전자 cfg 변형.** 26.5 일 pseudo-sync 상태
  가 동역학적으로 예상되는 균형. 자유 회전자 대안 (P_rot ≈ 16 h,
  Neptune-analog) 이 cfg 변형으로 보존됨. 시각 차이는 띠 구조의
  sharpness 에 있음 — 자유 회전이 더 sharp 한 띠와 더 빠른
  substellar 드리프트를 줌.
- **직접 대기 특성화.** 2026 사이클까지 JWST 프로그램이 HD 69830 c
  를 타깃한 적 없음. 미래 직접 촬영 시설 (HabEx, LUVOIR, ELT/
  HARMONI) 이 ~15 milliarcsec 각도 분리에서 c 를 해상할 수 있음 —
  행성/별 명암비 ~10⁻⁸ 의 ELT 범위 안. cfg 의 띠 chromophore
  형태가 반사광 분광으로 검증 가능.
- **b 및 d 와의 공명 확인.** b:c 주기 비 31.56/8.667 = 3.64 는 4:1
  평균 운동 공명에서 멀고, c:d 비 197/31.56 = 6.24 도 비공명. 장기
  동역학적 안정성 논문 (Ji 2007, Payne 2009) 이 시스템이 이 비공명
  구성에서 안정함을 확인. cfg 변형 필요 없음.
- **조석 가열 교차 확인.** e = 0.13 + Neptune 급 Q 에서 채택한
  0.002 W/m² 내부 열 플럭스는 자릿수 추정. 더 엄밀한 조석-진화
  모델링 (Bolmont 2013 스타일) 이 이를 정련할 것.
