# Element plasma colors

Per-element flame test / low-temperature plasma colors. Replaces pixel-sampling of the Helmenstine 2017 chart (whose decorative gradient blurs the intended hex). Source: chemistry literature + NIST Atomic Spectra Database, with the Helmenstine chart as visual sanity check.

Canonical source: `db/refs/element_plasma_colors.yaml`. This file is generated by `scripts/refs/render_element_colors_doc.py` — do not edit by hand.

## Status legend

| Status | Meaning | Hex |
|---|---|---|
| visible | Documented flame/plasma emission color | required |
| no flame color | Element exists but no visible flame coloration | null |
| not visible to humans | Emission falls outside 380–780 nm | null |
| too radioactive | Flame test unsafe / not characterized | null |
| half-life too short | Synthetic, decays before observation | null |
| no published data | Literature silent | null |

Reentry plasma at 8,000–15,000 K differs from atomic flame emission — molecular bands (N2+, C2 Swan, CN violet) dominate. See [composition-color reference](../../.claude/skills/firefly-cfg/references/composition-color.md) for bulk-gas reentry colors used in Firefly cfg.
