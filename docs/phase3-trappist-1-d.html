개요

TRAPPIST-1 d 는 0.79 R⊕ 의 암석 행성으로, M8V 초저온 왜성 주위를 4.05일 주기로 공전합니다. 고전적 거주 가능 영역 안쪽 가장자리에 위치하며, 최근 JWST/NIRSpec PRISM 의 투과 분광 관측 (Piaulet-Ghorayeb 외 2025) 으로 대기에 강한 제약이 걸렸습니다. 태양계 유사 조성 대부분이 95% 이상 신뢰도로 배제되어, 남은 시나리오는 (A) 완전 진공 표면, (B) 극히 얇은 대기 + 종단선 부근 고고도 물얼음 구름 두 가지입니다.

NearStars 채택 시나리오 — (B) 극히 얇은 대기 + 종단선 물얼음 구름. JWST 3σ 상한 압력 (~0.01 bar) 으로 설정하고, Turbet 2023 GCM 이 예측한 종단선 고고도 수증기 응결 특징을 보존합니다. (A) 완전 진공 보다 시각적으로 구별되는 요소를 남기되, 관측 데이터와 일관성을 유지하는 절충안.

표면 합성

T-1 b · c 의 JWST 방출/투과 데이터 (Greene 2023, Zieba 2023) 가 어둡고 저알베도인 bare-rock 표면과 일관됨이 확립됐습니다. d 의 경우 직접 방출 데이터는 아직 없지만, Piaulet 2025 의 투과 분광 해석은 bare-rock 시나리오를 완전히 허용합니다. 따라서 d 도 같은 해석을 확장하되, b/c 보다 평형 온도가 낮아 야간면 휘발성 응결물이 가능성으로 남습니다.

색상 선택

호스트 별은 2566 K M8V 왜성으로 SED 가 1.1 μm 부근에서 정점, 가시광은 적색–주황색이 압도적입니다. 인간 관측자에게 표면 반사는 광물 본연의 색과 무관하게 적갈색으로 편향됩니다. 신선한 basalt 광물의 광대역 가시광 알베도 (≈0.05–0.12, Hapke 보정) 는 Mercury / Moon mare 가 가장 가까운 태양계 참조. #2a2018 (어두운 회갈색) + #5a3220 산화철 반점.

산화철 반점

장기간 조석 잠금된 암석형 표면에선 별빛 UV 가 도달하는 영역에 표면 산화가 진행됩니다 (Turbet 2018 의 일면 표면 광화학 논의 일반화). 일면 substellar 영역 중심 ±30° 안쪽에 강하게 편향된 산화 반점 분포를 가정. 낮은 확신도 — 직접 반사 분광은 아직 없습니다.

지형

세 가지 요인이 표면 텍스처에 기여합니다. (1) ≲500 Myr 의 마그마 바다 단계 (Piaulet 2025 §8) 가 남긴 흐름 · 균열 구조, (2) ~8 Gyr 의 충돌 누적 (조석 잠금된 표면이라 leading hemisphere 에 충돌 편향), (3) 별-행성 자기 결합으로 인한 유도 가열 (Grayver 2022, Way 2025 인용) 가 제한적 화산 활동 지속 가능성. d 에 대해 (3) 의 직접 증거는 없으므로 기본은 (1)+(2) — Lunar highlands 정도의 충돌 분화구 밀도 + 일면 종단선 부근 마그마 잔재 구조.

대기 합성

JWST NIRSpec/PRISM 이 두 번의 통과에서 얻은 투과 스펙트럼은 항성 오염 보정 후 ±100–150 ppm 이내로 평탄합니다 (Piaulet-Ghorayeb 2025). 이로 인해 다음 시나리오들이 배제됩니다.

NearStars 선택 근거

"극히 얇은 대기 + 종단선 H₂O 얼음 구름" 합성을 채택합니다.

  • 압력 은 JWST 3σ 상한 (~0.01 bar = 1000 Pa) 으로 — 관측과 호환되는 가장 두꺼운 대기.
  • 조성 은 N₂ 우세 + 미량 CO₂ + H₂O. CH₄ · NH₃ 는 광분해 (Turbet 2018), H₂ 는 형성 + 탈출 (Hori & Ogihara 2020) 로 배제. CO₂ 미량은 외기 배출 모델 (Liggins 2021, Salman 2024) 이 지지.
  • 종단선 구름 은 전면 overcast 가 아닌 산발적 고고도 패치 — Turbet 2023 GCM 예측 일치, 진공 표면과 시각적 구별 제공.

자전 · 자전축

TRAPPIST-1 d 의 매개변수는 강한 동기 구성을 강제합니다. 이심률 0.00638 (Agol 2021), 자전축 기울기는 ≳7 Gyr 시스템 나이 동안 조석으로 0 으로 댐핑 (Agol 2021 §6.2), 자전-공전 공명은 1:1 이 가장 그럴듯합니다. Way 2025 는 3:2 가능성도 공식 배제되지 않음을 언급 (Correia 2014, Makarov 2018, Revol 2024) 하고 GCM 시뮬레이션 두 개를 3:2 가정 하에 수행. NearStars 는 기본을 1:1 로 채택; 3:2 대안은 후속 cfg variant 후보.

KSP 구현

완전한 1:1 조석 잠금 → 자전 주기 = 공전 주기 = 4.0508일. Kopernicus 의 rotationPeriod 를 궤도 period 와 동일 (349,989 초) 로 설정.

계절 없음

자전축 기울기 0 + 이심률에 의한 일조 변동 0.4% 미만 → 표면 좌표계에서 substellar point 고정. 표면 어느 지점이든 주야 사이클 없음, 계절 극지 이동 없음, 종단선 표류 없음.

시각 스타일링

표면과 대기 결정 사항을 종합한 렌더링 가이드.

전역 팔레트

어두운 basalt 본체 (#2a2018 기본, #5a3220 강조) 가 강렬한 적-주황 별빛 아래에서 깊은 황토색–숯색 행성으로 보입니다. 녹색 · 청색 채널 거의 없음. 가장 큰 시각적 콘트라스트는 종단선에서의 그레이징 조명이 표면 기복을 드러낼 때 발생.

일면 (dayside)

밝은 substellar 영역 (~390 K, ~118 °C), 천정에 가까운 조명이 그림자를 평탄화해 특징이 다소 약함. 산화철 반점은 substellar 점 ±30° 안쪽에서 강함.

종단선 띠

가장 사진찍기 좋은 구역. 고고도 물얼음 구름이 그레이징 별빛을 잡아 적색 표면 위에 희미한 청백 잔물결로 나타납니다. 상당한 지형 그림자 콘트라스트.

야간면 (nightside)

차갑고 (~150 K) 어둡습니다 — 가시광 특징 없이 열적외선 방출만. KSP 야간면 조명은 매우 낮은 ambient 로 설정 — 유일한 광원은 TRAPPIST-1 b · c (드문 가까운 합) 또는 외행성들의 반사광.

대기 헤이즈

0.01 bar 대기는 거의 보이지 않습니다. 행성 limb 의 우주 배경 대비로만 보이는 얇은 (3–5 km) 옅은 은청색 limb 헤이즈 (#a8b0c0).

하늘에서의 별

TRAPPIST-1 은 d 의 하늘에서 약 2.85° (지구에서 본 태양의 약 5.7배 각크기). 깊은 적-주황색 (≈2566 K 흑체 → #ff7a1a 부근). 표면 밝기는 화성에서 본 태양과 비슷 (~1.12 S⊕). 1 μm 이상 파장에선 별의 항성 플레어가 가시적 (Howard 2023, Piaulet 2025 §1 인용) — KSP 에서는 가끔의 플레어 광량 깜빡임으로 표현 가능 (옵셔널).

하늘의 형제 행성들

내합 시 TRAPPIST-1 c 는 ~0.5° 각크기 (지구의 달 정도), e 는 ~0.3°. 거의 동일 평면 공전 → 며칠마다 합 발생.
