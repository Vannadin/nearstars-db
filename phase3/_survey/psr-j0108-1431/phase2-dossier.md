# Phase 2 dossier — PSR J0108−1431 (nearest radio pulsar)

Solitary, ordinary (non-recycled), old/faint radio pulsar near the death
line. Constellation Cetus. Gathered from literature (agent run 2026-06-03).

## Disputed / flagged
- **Distance** ~85–130 (old DM) → 240 (raw VLBI) → **210 pc (best)**. Parallax only ~3σ.
- **Ṗ/τ/Ė/B**: Pavlov-literature vs ATNF timing solutions differ ~1.4–2×. Cite Pavlov set, note ATNF.
- **X-ray**: thermal vs non-thermal unresolved (low counts); Chandra kT 0.28 vs XMM 0.11 keV.
- **No mass/radius** measured — canonical only.
- Nearest NS if pulsar requirement dropped: RX J1856.5−3754, ~123 pc (radio-quiet, "Magnificent Seven").
