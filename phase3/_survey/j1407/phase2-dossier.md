# Phase 2 dossier — J1407b / V1400 Cen (largest known ring system)

Single-eclipse (2007) "super-Saturn." Host is a genuine young PMS K5 star;
the companion + rings are a compelling but UNCONFIRMED, model-dependent,
single-event interpretation that later work casts doubt on. Constellation
Centaurus. Gathered from literature (agent run 2026-06-03).

## Status & caveats — READ
- **Robust:** the 2007 eclipse light curve is real SuperWASP/ASAS data (deep ≳3 mag, ~54–56 d, centered 2007-04-29); host star well-characterized young PMS K5 @ Gaia 138 pc.
- **Tentative/disputed:** rings are best-fit MODEL of one light curve, not imaged. **Only one eclipse ever seen.** Mentel 2018 (plates 1890–2018) found NO confirming dimming → "cannot confirm the 2007 event was a bound object." 2019 monitoring: no trace. Mass spans ~10×. Period unconfirmed. Sutton 2019: exomoon-resonance gap origin unlikely. **Alternative: a free-floating disked object that coincidentally transited V1400 Cen** (would dissolve the bound-planet picture).
